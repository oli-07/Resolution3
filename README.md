Space Homepage

This is a browser homepage project made with HTML, CSS, and JavaScript.

It shows:
- A NASA picture as the background.
- The current time.
- Local weather.
- Cat Facts
- Sunrise and sunset times.

I used APIs to fetch live data and display it on the page.
 How to use it

Open `index.html` in your browser and allow location access so the time, weather, and sunrise/sunset features can work.

 Files

- `index.html` — page structure
- `main.css` — styling
- `script.js` — API calls and live updates
